# Lab_3_OS
Описание проекта
Это консольное многопоточное приложение, состоящее из основного потока (main) и нескольких рабочих потоков (marker). Программа демонстрирует работу с разделяемыми ресурсами (массивом) в многопоточной среде с синхронизацией потоков.

Функциональность
Основной поток (main)
Создает массив целых чисел заданного размера

Инициализирует массив нулями

Запускает указанное количество потоков marker

Управляет работой потоков marker:

Ожидает сигналы от потоков о невозможности продолжения работы

Отображает состояние массива

Запрашивает у пользователя номер потока для завершения

Управляет завершением и продолжением работы потоков

Поток marker
Генерирует случайные числа и пытается помечать элементы массива

Работает циклически, пока не встретит уже помеченный элемент

Сообщает основному потоку о невозможности продолжения работы

Ожидает команды от основного потока (продолжить или завершиться)

При завершении обнуляет все помеченные им элементы
